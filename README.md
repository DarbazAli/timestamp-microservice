# API Project: Timestamp Microservies
This project is an API Microserves that listenes in the following endpoint url:
```
GET /api/timestamp/:date_string?
```
And respondes back in the following format (if input is valid):
```
{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
```
See the project on [Repl.it](https://boilerplate-project-timestamp.darbazali.repl.co)
